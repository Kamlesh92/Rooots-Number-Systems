Binary Numbers

Definition: Binary number system uses only two digits: 0 and 1.

Base: 2

  • All digital computers operate using binary numbers.
  • Each digit is called a bit.
  • Binary number example: 1011.
  • Binary place values represent powers of 2 (e.g., 20, 21, 22, ...).

Highlights

  • Used internally by almost all modern computers and computer-based devices.
  • Easy to implement with digital electronic circuitry.
  • Binary addition follows rules similar to decimal but with base 2.
  • Conversion to/from decimal and other number systems is fundamental.
